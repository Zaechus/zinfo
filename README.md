- Android
- Linux

    - Arch Linux
    - Artix Linux
    - Debian
    - Fedora
    - Gentoo
    - Manjaro
    - NixOS
    - openSUSE Leap
    - openSUSE Tumbleweed
    - Ubuntu
    - Void Linux
- Windows
